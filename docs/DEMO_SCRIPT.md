# Fillosophy Demo Script (90-120 seconds)

**Preparation Note:**
> Test this full flow at least twice before recording to catch any last-minute bugs.

### Script / Flow

**1. [0:00-0:10] The Problem**
- Start recording on a blank, tedious internship application form (e.g., a dummy Google Form or a long Unstop application).
- Scroll through the empty fields quickly to demonstrate the friction of manual entry.

**2. [0:10-0:20] The Solution (Upload)**
- Click the Fillosophy extension icon in the Chrome toolbar.
- The popup opens smoothly on the **Upload** tab.

**3. [0:20-0:35] Profile Extraction**
- Drag and drop `sample_resume.pdf` into the designated dropzone.
- The UI updates to show the file is accepted.
- Click **Extract & Save Profile**. 
- Briefly show the loading state before the success message appears.

**4. [0:35-0:45] Data Verification**
- The UI automatically switches to the **Profiles** tab (or switch manually).
- Show the populated preview fields (Name, Email, CGPA, Degree, Skills), proving the data was extracted perfectly.

**5. [0:45-0:55] Context Switch**
- Close the extension popup (optional) and navigate to a real form on screen.
- E.g., Unstop, Internshala, or the prepared Google Forms test page.

**6. [0:55-1:05] Smart Matching**
- Open the Fillosophy popup again.
- Switch to the **Autofill** tab.
- Watch as the stats populate dynamically (e.g., "12 fields found", "11 high confidence", "1 needs review").

**7. [1:05-1:15] The Magic (Autofill execution)**
- Click the **Autofill This Form** button.
- The popup stays open or closes, and the fields in the background web page instantly fill with data.
- Notice the **green outlines** appearing around successfully filled fields.

**8. [1:15-1:25] Confidence Overlay**
- Scroll down to the bottom-right corner of the page to show the Fillosophy in-page overlay panel summarizing the fill results.

**9. [1:25-1:35] Reviewing Flagged Fields**
- Find a field highlighted with an **amber outline** (indicating low confidence / needs review).
- Correct it or accept it manually to demonstrate that the system empowers the user to verify edge cases.

**10. [1:35-1:45] Form Completion**
- Quickly scroll through the fully populated, clean form ready for submission.

**11. [1:45-1:55] Portability / Export**
- Open the extension one last time, go to the **Profiles** tab.
- Click **↓ Export JSON** to download the profile.
- Show the downloaded file appearing in Chrome's download bar (demonstrating data ownership and portability).

**12. [1:55-2:00] Outro**
- End the recording focused on the beautifully filled form, fading out.
